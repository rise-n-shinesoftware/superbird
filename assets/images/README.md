# Superbird Game Assets

This folder contains the visual assets for the Superbird game. To make the game objects more realistic, replace these placeholder files with actual images.

## Required Images:

### Bird Sprites
- `bird.png` - Main bird sprite (32x32 or 64x64 pixels)
- `bird_flap.png` - Bird with wings up (optional animation frame)

### Obstacle Sprites
- `pipe_top.png` - Top pipe obstacle
- `pipe_bottom.png` - Bottom pipe obstacle
- `pipe_body.png` - Middle section of pipes (tileable)

### Power Pickup Sprites
- `power_red.png` - Red burst power pickup
- `power_blue.png` - Blue slowmo power pickup
- `power_green.png` - Green shield power pickup
- `power_yellow.png` - Yellow multiplier power pickup
- `power_purple.png` - Purple blink power pickup
- `power_black.png` - Black phase power pickup

## Image Specifications:
- Format: PNG with transparency
- Size: 32x32 to 64x64 pixels recommended
- Style: Pixel art or realistic sprites that fit the game's aesthetic

## 3D/Realistic Look:
For a more 3D/realistic appearance, consider:
- Isometric sprites
- Drop shadows
- Lighting effects
- Texture mapping
- Multiple animation frames

Place your image files in this `assets/images/` folder and they will automatically be loaded by the game.
